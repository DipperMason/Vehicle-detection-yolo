## 📖 О проекте

Простой и эффективный инструмент для обнаружения транспортных средств на изображениях. Проект объединяет обучение модели YOLOv8 и детекцию в одном файле. Достаточно запустить скрипт, выбрать изображение и получить результат с визуализацией и оценкой плотности трафика.

### ✨ Что умеет

| Функция | Описание |
|---------|----------|
| 🎯 **Обучение** | Автоматическое обучение YOLOv8 на вашем датасете |
| 🔍 **Детекция** | Находит все автомобили на изображении |
| 📊 **Анализ** | Подсчитывает количество и оценивает плотность |
| 🖼️ **Визуализация** | Рисует рамки и выводит статистику |
| 📈 **Метрики** | Сохраняет результаты обучения в Excel |

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/DipperMason/vehicle-detection-yolo.git
cd vehicle-detection-yolo
2. Установите зависимости
bash
pip install -r requirements.txt
3. Подготовьте датасет
Создайте файл data.yaml с путями к вашему датасету:

yaml
train: ./path/to/train/images
val: ./path/to/val/images
nc: 1
names: ['vehicle']
4. Запустите программу
bash
python main.py
Программа автоматически:

✅ Обучит модель (20 эпох)

✅ Сохранит метрики в Excel

✅ Откроет окно для выбора изображения

✅ Покажет результат с обнаруженными автомобилями
